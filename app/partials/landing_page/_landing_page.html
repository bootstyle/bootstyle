About Bootstyle

What it Bootstyle?

Bootstyle is a full featured Twitter Bootstrap customizer. It's like a Bootstrap theme roller on steroids. We're bringing the desktop design tools you know and love to Twitter Bootstrap customization on the web. Things like sliders, color pickers, and so much more.

Bootstyle aims to be a Bootstrap customizer for developers and designers alike. It has been carefully crafted with so much love by developers and designers who love Bootstrap.

Why Bootstyle?

We built Bootstyle because we don't design great we pages in text editors. We use design tools. Bootstrap deserves a design tool. One built by designers and developers. So, here it is. We hope you love it as much as we do!

Features

Live Preview

See your changes as you make them. No browser refreshes here. No plugins or extensions required, either. Just tweak the controls and watch your creative genius unfold right in front of you.

Multi-Browser Sync

This feature is currently under development. Stay tuned and be the first to witness the glory!

Powerful Controls

  • Simple to use, powerfully implemented. Get more done and be happier doing it.
  • Color pickers
  • Super smart text inputs that process:
    • 3 or 6 digit hex colors: #000, #123456...
    • CSS color names: red, lightcoral, cornflowerblue...
    • Bootstrap LESS variables: @brand-primary, @body-bg...
    • CSS color functions: rgb(), rgba(), hsl(), hsla()
    • Any LESS function: lighten(color, amount), mix(color1, color2)...
  • Readout badges
  • Color swatches
  • Range sliders
  • Macro functionality built into to many controls.

HTML Editor

Sick of lorem ipsum? Wish you could see that template applied to your actual content? Now you can, just hit 'Edit HTML' and paste in your own template. Your changes are immediately available when you switch back to preview.
