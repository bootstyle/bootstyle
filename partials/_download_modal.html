DOM Setup Required

In order to get an identical theme to the one you are viewing, you need to make the following changes to your HTML.

Head

Add this to your <head>:

<link rel="stylesheet/less" href="path_to/bootstrap.less">
<link rel="stylesheet/less" href="path_to/theme.less">
<link rel="stylesheet/less" href="path_to/bootstyle.less">

Containers

All containers need this class:

<div class="{{ bootstyle.settings.grid_container_class }}">...</div>
